- Q: What is credentials in fetch API?<br>
It's an option from Fetch API that controls browser-managed authentication data.
Including: cookies (csrftoken, sessionid), http authentication entries, TLS certificats. It differs from headers. Headers option controls explicit values you add to request.
- Q: How is CSRF generated on the frontend?<br>
It's not, it is generated by django's CSRF middleware and sent to frontend (react).
